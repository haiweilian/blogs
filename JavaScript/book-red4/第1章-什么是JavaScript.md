## JavaScript 实现

JavaScript 的实现包含三部分，分别是：`ECMAScript`（核心）、`DOM`（文档对象模型）、`BOM`（浏览器对象模型）。

- `ECMAScript` 是对语言规范的称呼，分别描述了：语法、类型、语句、关键字、保留字、操作符、全局对象。

- `DOM（Document Object Model）` 用于操作 `HTML` 的应用编程接口。

- `BOM（Browser Object Model）` 用于支持访问和操作浏览器的窗口。
